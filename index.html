WINDOW

DOCUMENT

The window object represents the browser window or tab in which the web page is loaded.
The document object represents the HTML document loaded in the browser window or tab.
It acts as the global object in the browser's JavaScript environment and provides access to various properties and methods.
It is a property of the window object and provides access to the structure and content of the document
It encompasses the entire browser window and includes features like navigation, location, history, timing, and more.
It allows manipulation of the document's elements, such as selecting elements, modifying their properties, adding or removing elements, etc.
Examples of properties and methods available through the window object are window.innerWidth, window.alert(), window.setTimeout(), etc.
Examples of properties and methods available through the document object are document.getElementById(), document.createElement(), document.querySelector(), etc.
